📖 매매 전략

1장대양봉 + 대량거래 = 세력 진입
2거래량 급감 음봉 = 매도세 소진
3이평선 위 지지 = 건전한 조정
4분할 매수 → 기준봉 고가 목표

📖 종가베팅 매매 원칙

1신고가 돌파: 52주 고점 + 장대양봉 확인
2눌림목 타점: 거래량 30% 이하 짧은 음봉
3수급 및 시황: 기관 순매수 유입 및 상승장 확인
4칼손절: 5일선 이탈 시 뒤도 돌아보지 않고 매도

📖 내일급등 선취매 전략

1자금 유입: 10일내 10% 이상 급등한 기준봉 확인
2이평 수렴: 5일선과 20일선의 간격이 좁아지는 구간
3볼밴 밀집: 볼린저밴드 상단 바로 아래에서 에너지 응축
4폭풍 전야: 전일 대비 거래량이 급감한 단봉 캔들 확인
